# Usage

-   set `enabled` in setting.json (`{bdsxDir}/plugins/ostag/setting.json`, ignore or remove `{bdsxDir}/ostag/setting.json`)
-   if true, after players joined, players' device os will be shown under their nametag

```json
{
    "enabled": true
}
```
